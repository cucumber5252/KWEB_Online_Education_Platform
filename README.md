# KWEB_Online_Education_Platform

## 상세 설명
고려대학교 정보대학 웹개발 동아리 KWEB 활동 기간 내부 테스트를 위해 24시간 동안 구현한 Online Education Platform 데모 버전입니다.

## 개발 기간

2024.09.08

## 주요 스택
- 프론트엔드: React.js
- HTTP 클라이언트: Axios
- 라우팅: React Router
- 스타일링: Styled-components

- 백엔드: Node.js (NestJS)
- 데이터베이스: PostgreSQL
- ORM: TypeORM
- 인증: JWT (JSON Web Token)
